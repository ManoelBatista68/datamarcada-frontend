chat WhatsApp — Meta Business
Phone Number ID
info
Encontre em: Meta for Developers → Seus Apps → [seu app] → WhatsApp → Configuração da API → campo "Phone Number ID".
Token Permanente
info
Gere em: Meta for Developers → Configurações do Negócio → Usuários do Sistema → Adicionar → gere token → selecione o app e marque a permissão "whatsapp_business_messaging". Use um token permanente, não temporário.
Atenção: o token é visível apenas no momento de geração. Guarde uma cópia segura.
Token de Verificação do Webhook
info
Crie uma senha de sua escolha (ex: datamarcada2024). Você vai inserir esse mesmo valor no campo "Token de verificação" ao configurar o Webhook no painel da Meta.
URL do Webhook (Callback URL)
info
Copie esta URL e cole no campo "URL de callback" em: Meta for Developers → WhatsApp → Configuração → Webhook. Assine o evento "messages".
Modo de Operação
info
"nativo" — usa o bot integrado do DataMarcada. "n8n" — desativa o bot para uso de automação externa via n8n. "desativado" — silencia todas as respostas automáticas.
dynamic_form Formulário de Cadastro (Meta Flows)

Quando um cliente sem cadastro tentar agendar, o bot enviará este formulário interativo. Para criar um Flows, acesse Meta Business Suite → WhatsApp Manager → Flows → [+] Novo.

ID do Flows
info
Encontrado em WhatsApp Manager → Flows → selecionar o flow → copiar ID. Deixe vazio para desativar o formulário automático.
Texto do Botão (CTA)
info
Texto exibido no botão da mensagem do bot. Máx. 20 caracteres.
ID da Tela Inicial do Flows
info
Deve corresponder ao campo "id" da primeira tela no JSON do Flows (ex: "TELA_CADASTRO").
payments Mercado Pago
Token de Produção (Access Token)
info
Encontre em: Mercado Pago Developers → Suas Integrações → [sua integração] → Credenciais de Produção → Access Token. Certifique-se de usar credenciais de PRODUÇÃO, não de teste.
Atenção: use credenciais de produção. Credenciais de teste não geram cobranças reais.
URL de Notificação IPN
info
Copie esta URL e configure em: Mercado Pago Developers → Suas Integrações → [sua integração] → IPN → URL do site em produção. Selecione o evento "Pagamentos".
Exigir Pagamento Antecipado
info
Quando ativado: o cliente recebe um link do Mercado Pago ao agendar e o status fica "Pagamento Pendente" até a confirmação. Quando desativado: o agendamento é confirmado diretamente sem cobrança.
Exigir pagamento via Mercado Pago ao agendar
Desative para clínicas que preferem cobrar presencialmente ou por outro meio.
open_in_new Acesso ao App
Link de acesso ao App
info
Link enviado ao cliente após listar consultas. Aparece no rodapé das mensagens "Minhas Consultas" e "Consultas Realizadas".
O usuário de acesso é o email cadastrado.
timer Comportamento do Bot
Delay inicial (segundos)
info
Tempo de espera antes de enviar a primeira mensagem de cada resposta. Simula o bot "digitando". Padrão: 3 segundos.
Valor mínimo aplicado a todas as mensagens da sessão.
Velocidade de leitura (palavras/min)
info
Define o tempo de espera entre mensagens consecutivas, baseado na quantidade de palavras da mensagem anterior. Padrão: 200 palavras/min. Menor valor = mais tempo para ler.
Ex.: 200 palavras/min → mensagem de 20 palavras gera 6s de espera (mínimo: delay inicial).
Horas de antecedência (Apresentar Agenda)
info
Horários de agenda com menos de X horas para o início são removidos automaticamente. Ex.: 4 = remove horários com menos de 4h de antecedência. Padrão: 4h.
Define também o corte mínimo para exibir horários disponíveis no bot.
notifications_active ⏰ Alerta de Consultas à confirmar
info
Informe as horas de antecedência separadas por vírgula. Ex.: 48,24,1 = enviar alerta 48h, 24h e 1h antes da consulta. Deixe em branco para desativar.
Use vírgula para múltiplos alertas. Requer templates aprovados: lembrete_consulta_presencial2 e lembrete_consulta_online2.
info
Horas após o agendamento para lembrar o cliente do pagamento pendente. Ex.: 1,6,12 = lembrar 1h, 6h e 12h após a consulta ser marcada. Só funciona com "Exigir pagamento". Máx. 24h (janela WhatsApp).
Enviado como mensagem comum (dentro da janela de 24h do WhatsApp). Não requer template.
